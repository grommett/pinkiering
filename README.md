
# PinkieRing [![Build Status](https://snap-ci.com/grommett/pinkiering/branch/master/build_image)](https://snap-ci.com/grommett/pinkiering/branch/master)

Portfolio site.  
http://pinkiering.com

### Install
```shell
$ npm install
```

### Development
To build files and serve them (files served from ```./dev```):
```shell
$ npm run dev
```
## Test
```shell
$ npm test
```
## Deployment
```shell
$ npm deploy
```
